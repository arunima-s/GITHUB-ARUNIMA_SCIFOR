1. What is HTML?
HTML stands for HyperText Markup Language and it is used to create
webpages. It uses HTML tags and attributes to describe the structure and
formatting of a web page.HTML consists of various elements, that are responsible for telling searchengines how to display page content. For example, headings, lists,images,links, and more.
